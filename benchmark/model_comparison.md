## Model Comparison for Audio Generation

Models are compared side by side comparing
1. average similarity between original text and transcribed text of generated audio using each model
2. average MOS score calculated using UTMOSv2
3. good ratio which is number of audio (generated by each model) labeled as "good"

For data generation for each language, we use the model that performed the best.
